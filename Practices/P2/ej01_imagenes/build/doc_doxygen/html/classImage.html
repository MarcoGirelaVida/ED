Detailed Description

T.D.A. Imagen.

Una instancia del tipo de dato abstracto Imagen nos permite almacenar imágenes de intensidades.

El TDA Imagen proporciona además distintas herramientas para la manipulación de dichas imágenes.

Para poder usar el TDA Imagen se debe incluir el fichero

#include <Imagen.h>

◆ ShuffleRows()

void Image::ShuffleRows ( )

Baraja pseudoaleatoriamente las filas de una imagen.

Precondition
rows < 9973
Postcondition
El objeto que llama al método contiene ahora una nueva imagen igual que la anterior pero con las filas ordenadas según el siguiente algoritmo: 𝑟^= (𝑟*𝑝) mod rows.
Donde 𝑟 es el nuevo índice de la fila 𝑟, 𝑝 es un coprimo de rows, y rows es el número de filas de la imagen. (Dos números son coprimos si no tienen ningún factor primo en común, por simplicidad usaremos 9973 como número primo por defecto)
Note
Sin terminar

◆ ValidSectionSmart()

bool Image::ValidSectionSmart ( int & nrow,
int & ncol,
int & height,
int & width ) const

Comprueba que la sección proporcionada se encuentra dentro de la imagen y si no lo está, a ser posible, los modificará para que lo estén.

Parameters
nrowReferencia a la variable que almacena la fila inicial para recortar
ncolReferencia a la variable que almacena la columna inicial para recortar
heightReferencia a la variable que almacena el número de filas
widthReferencia a la variable que almacena el número de columnas
Return values
Truesi la sección era válida o si ha sido posible modificar los valores para que lo sea.
Falsesi la sección es inválida y no ha sido posible modificarla para deje de serlo.
Postcondition
Si nrow > filas --> nrow = filas
Si height + nrow > filas -> height = filas-nrow
Si ncol > columnas --> ncol = columnas
Si width + ncol > columnas --> width = columnas-ncol
Note
No será capaz de arreglar la sección si nrow && height son valores inválidos ( ValidRow) o ncol && width son valores inválidos ( ValidCol)
